Detailed Description

Vocabulary Entry Class.

In this program, each vocabulary has its own Voca class, which has
vocabulary meta-data. VocaEngine handle all process using this Voca
Class. It is a data-based class. 'exp' & 'level' are informations
about user's frequency of the vocabulary. Higher 'level' means that
user remember this word well. Answering collectly in test, user can
gain 'exp' score from VocaEngine, and this score upgrades Voca's level.
